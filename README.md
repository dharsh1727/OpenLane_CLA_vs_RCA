# OpenLane_CLA_vs_RCA using Sky130

This project presents a **comparative ASIC implementation** of:

- Ripple Carry Adder (RCA)
- Carry Lookahead Adder (CLA)
  
across:
- 4-bit designs
- 16-bit designs

using the **OpenLane RTL-to-GDS flow** on the **Sky130 PDK**.
